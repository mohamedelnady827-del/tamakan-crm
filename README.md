# Tamakan CRM Arabic Web App

مشروع React + Vite جاهز للنشر على Vercel.

## التشغيل محليًا
```bash
npm install
npm run dev
```

## البناء
```bash
npm run build
```

## النشر على Vercel
1. ارفع المشروع إلى GitHub
2. ادخل Vercel
3. اختر New Project
4. اختر المستودع
5. اضغط Deploy

## الملفات
- src/App.jsx : الواجهة الرئيسية
- src/styles.css : التصميم
- src/main.jsx : نقطة التشغيل
